Resizing the browser

When resizing the browser window: The row containing the city names breaks first. The row containing the letters remains intact longer. This behavior occurs because, on screens smaller than the "sm" breakpoint, will take up the full width of the column whereas with the "md" breakpoint, as the screen size increases, it will adjust its width to fit within the grid which are designed to be more responsive to larger screens.
